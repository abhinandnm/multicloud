# MultiCloud Platform

A personal project to build a multi-cloud deployment platform from scratch using Python and Flask.

## Current Features
- Validate GitHub repository URLs
- Verify repository existence using the GitHub API
- Clone public GitHub repositories
- Detect project type (Python, Node.js, Java)
- Fetch repository information
- Install project dependencies

## Planned Features

- Automatic application startup
- Docker support
- Deployment to AWS
- Deployment to Azure
- Deployment to Google Cloud
- Live deployment logs
- Web dashboard for deployments
- User authentication

## Status

 This project is currently under active development. Features may change frequently and the application is not yet production-ready.

## Tech Stack
- Python
- Flask
- Git
- GitHub API
- HTML
